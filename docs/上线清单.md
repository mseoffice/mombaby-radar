## 4. 回滚方案

| 场景 | 操作 |
|---|---|
| 代码 Bug | `git revert` 或切回上一稳定 tag（M3 `c618e7c`），重新构建部署 |
| 数据库迁移失败 | 恢复 MySQL 备份；`ddl-auto=validate` 下不自动变更 schema |
| DeepSeek API 不可用 | 切回 dev profile（Mock 降级），恢复后切回 prod |
| 基础设施故障 | docker-compose 重启；如 ES 挂 → `xpack.security.enabled=false` 单节点恢复 |
