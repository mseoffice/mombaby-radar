## 3. 评论收件箱（P4）

| # | 检查项 | 预期 | ✅/❌ |
|---|---|---|---|
| M1 | 手动粘贴录入 | 选择平台 + 粘贴文本 → 点击录入 → 列表出现 `NEW` 评论 | |
| M2 | 评论分析 | 点击「分析」→ 状态变 `ANALYZED`，显示分类/情绪/意图 + 2–3 条回复方案 | |
| M3 | 选定回复 | 点击「采用回复」→ 状态变 `REPLIED` | |
| M4 | PII 脱敏 | 评论文本含手机号（如 `13812345678`）→ 录入后变为 `<手机号已脱敏>` | |
